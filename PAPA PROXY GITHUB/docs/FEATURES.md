# PAPA PROXY Features

## Routing

- Per-domain routing through a PAC script
- Built-in routed presets for popular AI and media services
- Custom routed domains
- Browser toolbar states for disabled, searching, and connected modes

## Proxy selection

- Multiple enabled proxies in one list
- Autonomous search for the first proxy under `2000 ms`
- Immediate removal of failed or too-slow proxies from autonomous search
- Automatic switch when the active proxy degrades on a routed tab

## Proxy sources

- Country lists loaded from the Proxifly GitHub repository
- Raw proxy list import from pasted text
- Manual proxy entry editing

## Favorites and saved lists

- Favorite proxy flag on the currently connected proxy
- Favorite-only search mode
- Continue search starting from the next proxy
- Saved proxy lists in extension storage
- Local JSON backup export for saved lists and favorites

## Testing and diagnostics

- Individual proxy testing
- `TEST ALL`
- Per-proxy latency, IP, and country result display
- Debug log panel inside settings

## User interface

- English and Russian localization
- Localized toolbar tooltip text
- Version and changelog screen inside the popup
